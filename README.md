# Habit Tracker

A React-based habit tracking application built with Vite.

## Features

- Add and manage daily habits
- Track habit completion progress
- Persistent storage using LocalStorage

## Project Structure

```
src/
├── components/
│   ├── Header.jsx
│   ├── HabitForm.jsx
│   ├── HabitList.jsx
│   ├── HabitListItem.jsx
│   └── ProgressActions.jsx
├── utils/
│   └── storage.js
├── constants.js
├── App.jsx
├── App.css
├── index.css
└── main.jsx
```
